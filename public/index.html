Welcome to Mockpass

Your ultimate Mock Server for Singpass & MyInfo Integration

🚀 Quick Start

Mockpass is currently running successfully! You can use this URL as the base endpoint for testing your application's Singpass authentication and MyInfo data retrieval.

To integrate, change your application's OIDC and MyInfo base URLs to point to this domain:

https://mockpass-sandy.vercel.app

👥 Persona Admin Dashboard

We provide a dedicated Admin interface to easily manage and generate mock user profiles (personas) for your testing scenarios.

You can dynamically configure their Date of Birth, Income, Education level, and other MyInfo attributes.

Go to Admin Dashboard

🔑 Authentication Flow

When your application redirects users to Mockpass to authenticate, they will be greeted with a beautifully redesigned login interface simulating the Singpass experience.

Users can log in using the NRIC format (e.g., S1234567A) and their generated Password to instantly retrieve their configured MyInfo profiles.
